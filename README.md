`partial_products.go`

> Asked by Google

Given a sorted set of integers [1, 2, 3, 4], produce a set of partial products such that each index contains the product of the set, sans the current index.

**Restrictions**
1. You cannot use division
2. Solution must be O(n)

**Example**
Input:  [1, 2, 3, 4]
Output: [24, 12, 8, 6]
